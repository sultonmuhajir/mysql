BAB 1: Pengantar MySQL

1.1. Pengenalan MySQL

Apa itu MySQL?

MySQL adalah sebuah sistem manajemen basis data relasional (RDBMS) yang menggunakan SQL (Structured Query Language) sebagai bahasa pengelolaan data. MySQL bersifat open-source, sehingga dapat digunakan secara bebas dan dimodifikasi sesuai kebutuhan pengguna. MySQL biasanya digunakan untuk mengelola data dalam aplikasi web, seperti situs e-commerce, sistem manajemen konten (CMS), dan aplikasi berbasis data lainnya.

Mengapa Memilih MySQL?

  • Kinerja Tinggi: MySQL dikenal karena kecepatan dan kemampuannya untuk menangani volume data yang besar dengan efisien.
  • Keandalan: MySQL memiliki reputasi yang baik dalam hal stabilitas dan keandalan.
  • Skalabilitas: MySQL dapat dengan mudah diskalakan dari aplikasi kecil hingga sistem yang sangat besar.
  • Komunitas Besar: MySQL memiliki komunitas pengguna dan pengembang yang besar, sehingga dukungan dan sumber daya tersedia dengan melimpah.


1.2. Sejarah dan Perkembangan

Sejarah Singkat MySQL

MySQL pertama kali dikembangkan oleh MySQL AB, sebuah perusahaan Swedia, pada tahun 1995 oleh Michael "Monty" Widenius, David Axmark, dan Allan Larsson. MySQL awalnya dikembangkan sebagai alternatif cepat dan andal untuk sistem basis data komersial.

Pada tahun 2008, MySQL AB diakuisisi oleh Sun Microsystems, yang kemudian diakuisisi oleh Oracle Corporation pada tahun 2010. Sejak saat itu, Oracle telah mengelola dan terus mengembangkan MySQL, merilis versi terbaru yang menambahkan fitur-fitur baru dan perbaikan.

Versi-Versi Utama MySQL

  • MySQL 3.x: Versi awal yang menyediakan fitur dasar RDBMS.
  • MySQL 4.x: Memperkenalkan fitur seperti query caching dan penggunaan SSL untuk keamanan.
  • MySQL 5.x: Menambahkan dukungan untuk stored procedures, triggers, views, dan informasi skema.
  • MySQL 8.x: Merupakan versi terbaru yang menawarkan peningkatan kinerja, keamanan, dan fitur-fitur baru seperti window functions dan CTE (Common Table Expressions).


1.3. Instalasi MySQL

MySQL dapat diinstal di berbagai sistem operasi seperti Windows, macOS, dan Linux. Berikut adalah langkah-langkah instalasi untuk setiap sistem operasi:

Instalasi di Windows

  1. Unduh Installer: Kunjungi MySQL Downloads dan unduh MySQL Installer untuk Windows.
  2. Jalankan Installer: Klik dua kali pada file installer yang diunduh dan ikuti petunjuk instalasi.
  3. Pilih Jenis Setup: Pilih jenis instalasi (Developer Default, Server Only, Full, Custom, dll.).
  4. Konfigurasi Awal: Tentukan root password dan pilih tipe server yang sesuai (Development Machine, Server Machine, Dedicated MySQL Server).
  5. Selesaikan Instalasi: Klik "Execute" untuk memulai instalasi dan tunggu hingga selesai.

Instalasi di macOS

# Install Homebrew (jika belum terpasang)
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install MySQL
brew update
brew install mysql

# Memulai MySQL Server
brew services start mysql

# Mengatur password root
mysql_secure_installation

Instalasi di Linux (Ubuntu)

# Update sistem paket
sudo apt update

# Install MySQL Server
sudo apt install mysql-server

# Mengatur password root
sudo mysql_secure_installation


1.4. Konfigurasi MySQL

Setelah instalasi, MySQL perlu dikonfigurasi untuk memastikan keamanan dan kinerja optimal. Berikut adalah beberapa langkah konfigurasi dasar yang penting dilakukan:

  1. Mengatur Password Root: Password root adalah kunci utama untuk mengakses MySQL. Mengubah password root dari nilai default atau password awal yang telah ditetapkan sangat penting untuk mengamankan akses ke database.

  2. Mengamankan Instalasi MySQL: MySQL menyediakan skrip mysql_secure_installation yang dapat digunakan untuk meningkatkan keamanan dengan menghapus pengguna anonim, menonaktifkan login root dari jarak jauh, dan menghapus database tes yang tidak diperlukan.

  3. Mengatur Bind Address: bind-address digunakan untuk menentukan alamat IP yang diizinkan untuk mengakses server MySQL. Dengan mengatur bind-address ke 127.0.0.1, kita membatasi akses hanya dari localhost, sehingga memperkuat keamanan.

  4. Memulai dan Menghentikan MySQL: Kemampuan untuk memulai, menghentikan, atau me-restart layanan MySQL sangat penting untuk manajemen server. Ini memungkinkan administrator untuk melakukan pembaruan atau pemecahan masalah.

  5. Memeriksa Status MySQL: Memeriksa status MySQL memastikan bahwa layanan berjalan dengan benar. Jika ada masalah, administrator dapat dengan cepat mengidentifikasi dan mengambil tindakan yang diperlukan.

# 1. Mengatur Password Root
ALTER USER 'root'@'localhost' IDENTIFIED BY 'new_password';

# 2. Mengamankan Instalasi MySQL
sudo mysql_secure_installation

# 3. Mengatur bind-address
[mysqld]
bind-address = 127.0.0.1

# 4. Memulai dan Menghentikan MySQL
   # Untuk Windows:
   net start mysql
   net stop mysql

   # Untuk Linux/macOS:
   sudo systemctl start mysql
   sudo systemctl stop mysql
   sudo systemctl restart mysql

# 5. Memeriksa Status MySQL
sudo systemctl status mysql


1.5. Tools dan Interface

Untuk mengelola dan berinteraksi dengan MySQL, ada beberapa tools dan interface yang bisa digunakan, baik berbasis GUI (Graphical User Interface) maupun CLI (Command Line Interface). Berikut ini adalah beberapa tools populer yang sering digunakan.

  • MySQL Command Line Client (CLI): Antarmuka berbasis teks bawaan yang memungkinkan pengguna menjalankan perintah SQL langsung di terminal atau command prompt.
  • MySQL Workbench: Alat GUI resmi dari Oracle yang menyediakan fitur untuk administrasi server, desain database, dan eksekusi query SQL.
  • phpMyAdmin: Aplikasi berbasis web yang populer untuk mengelola MySQL melalui antarmuka web yang user-friendly dan mudah diakses.
  • HeidiSQL: Alat manajemen MySQL yang ringan dan gratis untuk Windows, menawarkan fitur sederhana namun kuat untuk administrasi database dan data.
  • DBeaver: Alat manajemen database multi-platform yang mendukung berbagai database termasuk MySQL, dengan fitur SQL editor dan administrasi server.
